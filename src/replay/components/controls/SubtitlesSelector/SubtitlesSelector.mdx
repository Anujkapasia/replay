# SubtitlesSelector

Control bar widget for controlling subtitle display and language selections.

## Summary

The subtitles selector lists the stream's available text tracks, and controls what track to display, if any. The control is only rendered if there are one or more text tracks.

The subtitles selector uses the `<Selector />` generic component, and is implemented similar patterns to `<AudioSelector />` and partly `<QualitySelector />`. Upon clicking on one selector item representing a track, it invokes `setProperties({ selectedTextTrack: track })` where `track` is the clicked selector item. The selector also contains one item for turning off display of subtitles. This item must be given a label through the `noSubtitlesLabel` prop. If the "No subtitles" item is selected, `setProperties({ selectedTextTrack: null })` is invoked.

## Example

<Playground>
  {() => {
    const textTracks = [
        { label: 'Finnish' , language: 'fi', kind: 'subtitles' }, 
        { label: 'German' , language: 'de', kind: 'subtitles' },
        { label: 'Spanish', languag: 'es', kind: 'subtitles' }
      ];
    return (
      <ShowCase height="90px" render={({ setProperties, currentTextTrack }) => (
        <SubtitlesSelector 
          setProperties={setProperties}
          currentTextTrack={currentTextTrack}
          noSubtitlesLabel="No subtitles"
          toggleContent={<Type/>}
          textTracks={textTracks}
        />
      )}/>
    );
  }}
</Playground>
