# ExitButton

Standalone overlay button for closing the player.

## Summary

Intended to be placed in one of the upper corners of the player. Is used in the default player for providing a way to shut down the player.

## Example

  <div style={{ backgroundColor: 'gray', height: '120px', position: 'relative' }}>
    <ExitButton content={<XCircle/>} onClick={() => console.log('Exit clicked.')} />
  </div>
