# AirPlayButton

Control bar button for invoking Safari's AirPlay device picker, allowing transferring the playback to a different device.

## Summary

In order for this button to render, the prop `isAirPlayAvailable` must have the value `true`. This property is reported from the video streamer based on browser feature detection and AirPlay target devices being available. Currently only Apple Safari (and iOS browsers built in on it) supports AirPlay.
 
The prop `isAirPlayActive` indicates to the button if picture-in-picture playback is currently active(`true`/`false`). This can be used for indicating the AirPlay mode in the button. Through the callback prop `setProperties` in invokes `setProperties({ isAirPlayTargetPickerVisible: true })` on clicks. Even if this is a toggle button, it cannot be used to hide the target picker. However, any other user gestures will do so.
