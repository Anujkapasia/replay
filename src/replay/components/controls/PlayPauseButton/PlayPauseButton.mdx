# PlayPauseButton

Control bar button for pausing or resuming playback. 

## Summary

The prop `isPaused` indicates to the button if the playback is paused, or playing (`false`). Through the callback prop `setProperties` in invokes `setProperties({ isPaused: value })` on clicks, with the opposite value of the `isPaused` prop passed as value.

## Example

<Playground>
  <ShowCase render={({ setProperties, isPaused }) => (
    <PlayPauseButton playingContent={<Pause/>} pausedContent={<Play/>} isPaused={isPaused} setProperties={setProperties} />
  )}/>
</Playground>
