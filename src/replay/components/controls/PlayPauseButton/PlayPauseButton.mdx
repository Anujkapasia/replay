# PlayPauseButton

Control bar button for pausing or resuming playback. 

## Summary

It sets the property `isPaused` on clicks, with the opposite value of the `isPaused` prop passed into the component.

## Example

<Playground>
  <ShowCase render={({ setProperties, isPaused }) => (
    <PlayPauseButton playingContent={<Pause/>} pausedContent={<Play/>} isPaused={isPaused} setProperties={setProperties} />
  )}/>
</Playground>
