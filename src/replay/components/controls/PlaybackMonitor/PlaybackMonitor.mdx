# PlaybackMonitor component

An overlay for displaying all current and previous video stream state properties. Can be used for debugging and monitoring.

## Summary

This is a convenience component not intended for end user cases.

## Example

The playback monitor can be displayed by pressing **Ctrl+Alt+V** in an actual rendering of the Replay player.
