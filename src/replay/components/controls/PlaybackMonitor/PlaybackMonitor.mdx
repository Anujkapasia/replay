# PlaybackMonitor component

An overlay displaying all current and previous video stream state properties. Can be used for debugging and monitoring.

## Summary

This is a convenience component not intended for end user cases. It is by itself a connected control.

## Example

The playback monitor can be displayed by pressing **Ctrl+Alt+M** in an actual rendering of the Replay player.
