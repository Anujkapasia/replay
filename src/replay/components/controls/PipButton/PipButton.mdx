# PipButton

Control bar button for toggling the browser's picture-in-picture mode for video playback.

## Summary

In order for this button to render, the prop `isPipAvailable` must have the value `true`. This property is reported from the video streamer based on browser feature detection.
 
The prop `isPipActive` indicates to the button if picture-in-picture playback is currently active(`true`/`false`). Through the callback prop `setProperties` in invokes `setProperties({ isPipActive: value })` on clicks, with the opposite value of the `isPipActive` prop passed as value.

## Example

    <PipButton isPipAvailable={true} pipActiveContent={<Square/>} pipInactiveContent={<Copy/>} isPipActive={isPipActive} setProperties={setProperties} />
  )}/>
