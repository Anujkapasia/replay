# GotoLiveButton

Control bar button for resuming after timeshifting to the live edge of the stream. Only rendering/relevant for live DVR streams.

## Summary

The GotoLiveButton is a toggle button, but it can only toggle if the prop `isAtLiveEdge` is `false`. When clicking the button in this case, it invokes `setProperties({ isAtLiveEdge: true })`. If the button prop `isAtLiveEdge` is `true`, the button can present a different state, and clicking it has no effect.

## Example

<Playground>
  <ShowCase render={({ setProperties, isAtLiveEdge }) => (
